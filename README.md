# VisualSort

This repo is configured to publish `index.html` to GitHub Pages automatically.

## One-time GitHub setup

1. Push this repo to GitHub on the `main` branch.
2. In GitHub, open **Settings → Pages**.
3. For **Source**, select **GitHub Actions**.
4. Push a commit (or run the **Deploy site to GitHub Pages** workflow manually).

After deploy finishes, your site will be available at:

- `https://<your-username>.github.io/visual-sort/`

## Local preview

Open `index.html` directly in your browser, or run a local static server if you prefer.
